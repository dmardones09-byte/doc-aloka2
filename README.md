**Bienvenido a Aloka2** RP V2

## Bienvenido a una ciudad viva

Este servidor está pensado para historias con peso.

Aquí cada decisión deja huella.

Tu personaje no solo entra al mapa.

Entra a un mundo con memoria, tensión y oportunidades.

{% hint style="success" %}
La meta no es ganar escenas.

La meta es construir una historia que valga la pena recordar.
{% endhint %}

### ¿Qué vas a vivir aquí?

* **Rol con intención.** Cada escena debe tener sentido.
* **Progresión orgánica.** Tu crecimiento se gana jugando.
* **Conflicto con consecuencias.** Lo que haces cambia relaciones.
* **Mundo compartido.** Tu historia impacta a otros jugadores.

{% tabs %}
{% tab title="🌆 Vida civil" %}
Levanta un negocio, crea contactos y construye reputación.

La ciudad también se define en lo cotidiano.
{% endtab %}

{% tab title="🚔 Rol legal" %}
Protege el orden, responde a crisis y participa en escenas de alto impacto.

La autoridad también crea narrativa.
{% endtab %}

{% tab title="🕶️ Bajo mundo" %}
Muévete con cuidado, gana territorio y juega con riesgo real.

El poder aquí no se regala.
{% endtab %}
{% endtabs %}
