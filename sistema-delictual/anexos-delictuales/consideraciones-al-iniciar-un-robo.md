# Consideraciones al Iniciar un Robo

## Consideraciones al Iniciar un Robo

{% hint style="info" %}
Este criterio ya fue integrado en [Actos Delictuales](../actos-delictuales.md).
{% endhint %}

### Antes de iniciar

Debes tener plan, salida, número razonable de participantes, objetivo claro y contar con policias necesarios.

### Evalúa

* horario.
* presencia policial.
* entorno civil.
* nivel de riesgo.
* capacidad real del grupo.

Abrir por impulso suele romper la escena.
