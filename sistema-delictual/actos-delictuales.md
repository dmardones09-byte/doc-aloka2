# Actos Delictuales

## Actos Delictuales

### Regla general

Todo acto delictual debe respetar cupos, armamento, vehículos y restricciones propias.

Siempre aplica el **+1** de **Policias** cuando corresponda.

{% hint style="info" %}
Esta página concentra los criterios operativos más usados en robos, secuestros y escenas delictuales.
{% endhint %}

### Antes de iniciar un acto

Debes tener:

* plan claro.
* salida prevista.
* número razonable de participantes.
* objetivo definido.

Antes de iniciar, evalúa:

* horario.
* presencia policial.
* entorno civil.
* nivel de riesgo.
* capacidad real del grupo.

Abrir una escena por impulso suele romper el rol.

### Condiciones generales

#### Siempre debes respetar

* proporcionalidad.
* preparación.
* continuidad.
* fair play.
* capacidad real del grupo.

#### No puedes

* encadenar delitos sin descanso para abusar del sistema,
* usar personajes secundarios como herramientas desechables,
* romper el ambiente del servidor por beneficio rápido.

El acto delictual válido deja historia, no solo loot.

### Negociación

La negociación debe tener intención real.

#### Está prohibido

* pedir imposibles,
* usar negociador falso sin control de escena,
* romper negociación para disparar gratis,
* alargarla solo para traer refuerzos ocultos.

Si negocias, debes sostener el rol con seriedad.

### Rehenes ficticios

* Los rehenes ficticios solo existen si el sistema o el Staff los valida.
* No pueden usarse para inflar exigencias o forzar ventajas absurdas.
* Si el rehén no aporta rol creíble, no se considera válido.

### Cacheo delictual

Cachear a alguien requiere control efectivo de la escena.

El cacheo en ciudad es válido cuando proviene de un acto delictual activo y justificado (por ejemplo, asalto, secuestro o guerra territorial), siempre fuera de zona segura.

#### No es válido

* cachear en pleno fuego cruzado,
* sacar objetos infinitos,
* asumir hallazgos instantáneos sin rol.

El tiempo y el contexto importan.

### Robos en fase

Algunos robos requieren progresión por etapas.

#### Debes respetar

* requisitos previos,
* tiempos,
* participantes permitidos.

#### Está prohibido

* saltar fases,
* iniciar sin preparación,
* repetir la misma actividad de forma abusiva para farmear.

### Robo a civiles

* **Policías mínimos:** 2 conectados.
* **Máximo civiles:** 4 delincuentes.
* **Máximo OD:** 12 delincuentes.
* **Vehículos:** 1 para civiles, 2 para OD.
* **Tirador:** no permitido.
* **Refuerzo:** no permitido.
* **Armamento:** cualquiera, según la zona.
* **Restricciones:** no se puede robar comida, agua ni herramientas de trabajo.
* No se puede asaltar a **EMS** ni **mecánicos** en servicio.
* La cobertura corresponde a **Carabineros de Alokados** según la zona.
* Solo se puede abatir si es necesario.

### Secuestro a civiles

* **Policías mínimos:** 4 disponibles.
* **Máximo civiles:** 4 delincuentes.
* **Máximo OD:** 12 delincuentes.
* **Vehículos:** 2 para civiles, 3 para OD.
* **Armamento:** cualquiera, según la zona.
* **Tirador:** no permitido.
* **Refuerzo:** no permitido.
* **Prohibiciones:** no puedes mantener a una persona secuestrada por más de **30 minutos** sin rol de peso.
* El rol puede extenderse hasta **1 hora** si está bien justificado.
* No se permiten roles sexuales ni agresivos sin consentimiento.
* La cobertura corresponde a **Carabineros de Alokados** según la zona.
* Solo se puede abatir si es necesario.

### Cosecha y proceso de drogas

* **Policías mínimos:** 5 conectados.
* **Civiles:** mínimo 1, máximo 4.
* **OD:** mínimo 1, máximo 16.
* **Vehículos:** 1 para civiles, 4 para OD.
* **Armamento:** cualquiera, según la zona.
* **Tirador:** no permitido.
* **Refuerzo:** no permitido.
* **Prohibición:** no se puede robar comida ni agua.
* Debes enviar el entorno de **proceso / cosecha** antes de comenzar.

### Venta de droga

* **Policías mínimos:** 2 disponibles.
* **Vehículos:** máximo 1.
* **Armamento:** bajo o medio calibre.
* **Tirador:** no permitido.
* **Refuerzo:** no permitido.
* El entorno debe detallar cantidad de personas, vestimenta, armamento y vehículo.
* La cobertura corresponde a **Carabineros de Alokados** según la zona.

### Robo a tienda 24/7

* **Policías mínimos:** 2 disponibles.
* **Civiles:** mínimo 1, máximo 2.
* **OD:** mínimo 1, máximo 4.
* **Vehículos:** 1 para civiles, 1 para OD.
* **Armamento:** bajo calibre o armas blancas.
* **Tirador:** no permitido.
* **Refuerzo:** no permitido.
* **Rehén:** se puede usar rehén de cajero solo si no existe otro rehén.
* **Negociación:** máximo **10 minutos**.
* Si la policía no llega tras **15 minutos** desde el entorno, los asaltantes pueden escapar.
* En ese caso deben enviar entorno de escape de robo a tienda.
